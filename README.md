# Laravel GraphQL Admin API

Production-ready GraphQL Admin API for E-commerce built with Laravel + Lighthouse + Sanctum.

## 🚀 Features

- Admin Authentication
- Product Management
- Category Management
- Brand Management
- Order Management
- Customer Management
- Inventory Control
- Dashboard Analytics
- Role & Permission
- GraphQL Pagination / Filtering / Search

## 🛠 Tech Stack

- Laravel
- GraphQL
- Lighthouse
- Sanctum
- MySQL
- PHP 8+

## 📦 Installation

```bash
git clone https://github.com/zillucse08/laravel-graphql-ecommerce.git
cd laravel-graphql-admin
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
